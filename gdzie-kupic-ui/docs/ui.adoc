= Gdzie Kupić UI

- Materia UI based components
- Minimalistic design
- User-friendly interface
- ChatGPT UI like 

== Components

=== Left Sidebar: Unauthenticated User
- Home button
- Post history button
- Register button
- Login button

=== Left Sidebar: Authenticated User
- Home button
- Post history button
- Logout button

=== Left Sidebar: Merchant Agent
- Home button
- Response history button
- Logout button

=== User Post Card

- Title
- Description
- Category
- Location
- View count
- Merchant responses count
- Time since posted

=== Merchant Response Card

- Merchant name
- Response content
- Time since response
- Viewed status

== Views

=== Home Page: User

- List of active posts
- Create Post button

=== Post Detail Page: User

- Post content
- List of merchant responses

=== Post Detail Page: Merchant
- Post content
- Option to respond to post
- Option to message user for more information before responding

=== Merchant Response Page: User

- Response content
- Merchant information
- Conversation interface with merchant 
- Option to accept response and close post
- Option to reject response and keep post open
- Option to message merchant for more information
